# Multiple Linear Regression 📈

A regression algorithm that predicts salary based on 
multiple input features using Multiple Linear Regression.

## Dataset
- **File:** Salary_Data.csv
- **Features:** Age, Years of Experience
- **Target:** Salary

## What's Covered
- Exploratory Data Analysis (EDA)
- Pairplot & Heatmap (correlation analysis)
- Train/Test Split (80/20)
- Model training using LinearRegression
- Coefficient & Intercept extraction

## Formula Used
```
y = m1*YearsExperience + m2*Age + c
y = 4882.14*YearsExperience + 2567.51*Age - 20612.69
```

- Pandas, Matplotlib, Seaborn
- Scikit-learn
